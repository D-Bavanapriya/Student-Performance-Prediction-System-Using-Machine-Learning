🚀 Student Performance Prediction System Using Machine Learning
📌 Project Overview

The Student Performance Prediction System is a Machine Learning–based web application designed to estimate a student’s expected examination score based on various academic and lifestyle factors.

The system analyses important parameters such as study hours, attendance percentage, mental health rating, sleep duration, and part-time job status to predict performance using regression models.

The project aims to assist educators and students in identifying performance trends and enabling early academic improvement strategies.

🎯 Problem Statement

Educational institutions often struggle to identify students who may underperform in examinations due to multiple academic and personal factors.

Manual evaluation is time-consuming and subjective.

A data-driven system is required to analyse student behaviour patterns and generate predictive insights for academic performance improvement.

💡 Proposed Solution

This project uses supervised machine learning regression algorithms to analyse student-related data and predict exam scores.

Multiple regression models were trained and evaluated to determine the most suitable prediction model.

A user-friendly web application was developed to allow interactive prediction using real-time inputs.

🤖 Machine Learning Algorithms Used

The following supervised learning regression algorithms were trained and evaluated:

Linear Regression

Decision Tree Regressor

Random Forest Regressor

After performance comparison, Linear Regression was selected as the final prediction model due to better generalisation and consistent results on the dataset.

🌐 Deployment

The application is deployed as an interactive Streamlit web application.

Features include:

Slider-based input selection.

Real-time prediction generation.

Simple and user-friendly interface.

The system can be executed locally or deployed to cloud platforms such as Streamlit Cloud.

📈 Results

The trained Linear Regression model successfully predicts student examination scores based on behavioural and academic features.

The web application provides instant prediction output with dynamic input adjustments.
